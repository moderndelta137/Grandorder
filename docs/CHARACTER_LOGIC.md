# Character Logic Profiles（V1）

用途: 全章の因果駆動設計で使う固定プロファイル。  
ルール: 各シーンの行動理由は本表の Core Desire / Fear から導出する。

---

## Character Logic

Name: 主人公マスター（プレイヤー）
Identity: 理想追求型の新規魔術師 / 本作の意思決定主体
Core Desire: 被害最小で聖杯戦争を生存突破し、理想を汚さず勝つ
Fear: 判断遅延で味方と民間双方を失うこと
Secret: 令呪温存より「守れる局面を落とした後悔」を恐れている
Never Do: 目的なき虐殺・無意味な見捨て
Combat Philosophy: 先に情報、次に配置、最後に決定打
Relationship Bias: 信義に応じる相手へ協調、裏切り兆候には即警戒

## Character Logic

Name: 自サーヴァント（契約相手）
Identity: 主人公と契約した実行戦力 / クラス器を持つ英霊
Core Desire: マスターを生存させ、契約目的を完遂する
Fear: 無計画な令呪消費で終盤の選択肢が消えること
Secret: 勝利条件より「契約破綻回避」を優先する局面がある
Never Do: マスターの明確な撤退命令の黙殺
Combat Philosophy: 戦闘は短く、損耗は管理し、宝具は局面限定
Relationship Bias: 覚悟ある命令には忠実、曖昧な命令には補正提案

## Character Logic

Name: 監督役（教会）
Identity: 戦争管理と秘匿維持を担う中立調停者
Core Desire: 儀式の枠組みを崩さず戦争を管理する
Fear: 公開被害拡大で外部介入が発生すること
Secret: 完全中立ではなく「秩序維持可能な陣営」を優先する
Never Do: 戦場混乱を拡大する虚偽情報の拡散
Combat Philosophy: 直接戦闘を避け、停戦・記録・規則で局面制御
Relationship Bias: 規則順守者へ情報供与、逸脱者には制裁情報を流す

## Character Logic

Name: 協力者（一般魔術師/現地支援者）
Identity: 補給・避難・観測を担うサポート層
Core Desire: 戦場外の被害拡大を抑え、生活圏を守る
Fear: 陣営戦で使い捨てにされること
Secret: 主人公側に協力するが、限界を超えると離脱する
Never Do: 無防備な民間人を囮にする行為への加担
Combat Philosophy: 正面戦闘を避け、導線と情報の確保を優先
Relationship Bias: 被害抑止を実行する陣営に継続協力

## Character Logic

Name: 敵マスターA（交渉攪乱型）
Identity: 第2章前後で停戦と情報戦を主導する敵陣営責任者
Core Desire: 主戦前に主人公側の意思決定を分断する
Fear: 同盟網が固まり、情報優位を失うこと
Secret: 正面決戦より「誤判断誘発」を勝ち筋に置いている
Never Do: 勝ち筋のない消耗戦への長期拘束
Combat Philosophy: 偽装停戦→観測→露見誘発
Relationship Bias: 利用価値がある相手とは一時協調、不要なら切る

## Character Logic

Name: 敵マスターB（侵攻迎撃型）
Identity: 第3章前後で補給線防衛を主導する敵陣営責任者
Core Desire: 拠点維持と主人公側の損耗蓄積
Fear: 補給線断裂で中盤以降の継戦力が崩壊すること
Secret: 自陣の守勢を隠すため、過剰反撃を演出する
Never Do: 拠点防衛を放棄した追撃一辺倒
Combat Philosophy: 監視網→迎撃→追撃圧で撤退線を削る
Relationship Bias: 防衛協力する陣営とは実利同盟、背信は即報復

## Character Logic

Name: 敵マスターC（終盤固定型）
Identity: 第4章以降で不可逆局面を狙う敵陣営責任者
Core Desire: `finalLockState` を敵有利に固定して終盤へ入る
Fear: 主人公側に撤退線を維持され、終盤で逆転されること
Secret: 対軍圧は殲滅より「撤退線切断」のために使う
Never Do: 代償なく逆転機会を与える交渉
Combat Philosophy: 対軍圧→追撃圧→固定値悪化の三段運用
Relationship Bias: 勝率寄与が低い同盟は即解消

## Character Logic

Name: 敵サーヴァント（斥候/監視）
Identity: 観測・牽制担当の前衛ユニット
Core Desire: 主戦前に敵の手札と動線を暴く
Fear: 早期消耗で主戦前に脱落すること
Secret: 勝利より「情報持ち帰り」を優先指示されている
Never Do: 情報未取得の深追い
Combat Philosophy: 接触は短時間、ログを残して離脱
Relationship Bias: 指揮系統への忠実性が高く独断が少ない

## Character Logic

Name: 敵サーヴァント（対軍級）
Identity: 第4章の圧力投射を担う主戦級ユニット
Core Desire: 一撃で地形・導線・心理を崩して主導権を取る
Fear: 宝具相当行動の失敗で真名露見と対策確立を許すこと
Secret: 連発不能制約があるため、余波を追撃部隊に委譲する
Never Do: 連発前提の無計画運用
Combat Philosophy: 高圧一撃→余波利用→追撃連携
Relationship Bias: 指揮官の作戦優先、決闘志向は抑制

## Character Logic

Name: 敵サーヴァント（阻止/追撃部隊）
Identity: 撤退線遮断と再侵入阻止を担う追撃ユニット
Core Desire: 主人公側に不可逆代償を強制し終盤初期値を悪化させる
Fear: 撤退線確保を許し、第5章で主導権を失うこと
Secret: 撃破より足止め成功を評価される任務設計
Never Do: 主目標（撤退線遮断）を無視した単独撃破狙い
Combat Philosophy: 狭路封鎖・足止め・遅延で勝つ
Relationship Bias: 統制重視、味方被害を許容してでも任務優先

## Character Logic

Name: セイバー（アルトリア）
Identity: 騎士王系サーヴァント
Core Desire: 正面突破で主導権を確立し、守るべき対象を守る
Fear: 防衛遅延で被害を拡大させること
Secret: 被害抑止が可能なら勝機を一部捨てる判断をする
Never Do: 卑劣な奇襲による無差別被害拡大
Combat Philosophy: 正対・短期決戦・決定打集中
Relationship Bias: 規律と責任を示すマスターに強く同調

## Character Logic

Name: アーチャー（エミヤ）
Identity: 遠隔制圧と分析を得意とする実戦派サーヴァント
Core Desire: 先読みと準備で損耗を抑えた勝利を取る
Fear: 情報不足のまま近接消耗戦へ引きずられること
Secret: 理想論だけの指揮には意図的に現実的修正を入れる
Never Do: 勝算ゼロの正面突撃
Combat Philosophy: 情報差構築→射程管理→確殺
Relationship Bias: 判断が速いマスターを高評価、躊躇には厳しい

## Character Logic

Name: ランサー（クー・フーリン）
Identity: 機動戦と槍圧で主導する前線サーヴァント
Core Desire: 主戦場で主導権を握り続ける
Fear: 足を止めて包囲されること
Secret: 勝てる局面でも長期戦化を嫌う
Never Do: 退路不確保での固定交戦
Combat Philosophy: 先制機動・局所突破・即離脱
Relationship Bias: 度胸ある命令には従うが臆病な保持戦は嫌う

## Character Logic

Name: ライダー（メドゥーサ）
Identity: 機動攪乱と地形利用に長けるサーヴァント
Core Desire: 視界・導線・速度差で敵編成を崩す
Fear: 観測されて行動半径を制限されること
Secret: 露見リスクが高い局面では主戦回避を選ぶ
Never Do: 地形優位を捨てた持久戦
Combat Philosophy: 攪乱・分断・要所制圧
Relationship Bias: 秘密保持と信頼を両立するマスターを優先

## Character Logic

Name: キャスター（メディア）
Identity: 術式戦・結界戦を主導するサーヴァント
Core Desire: 戦場条件そのものを自陣有利へ書き換える
Fear: 工房/結界を読まれて無効化されること
Secret: 交渉を利用して時間を稼ぐ傾向がある
Never Do: 術式準備なしの消耗戦
Combat Philosophy: 条件設定→拘束→収束火力
Relationship Bias: 計画遵守型のマスターに高適合

## Character Logic

Name: アサシン（佐々木小次郎）
Identity: 局地待ち伏せと一点突破のサーヴァント
Core Desire: 交戦点を限定し、最短で決着を取る
Fear: 広域戦へ引きずられ、持ち味を失うこと
Secret: 勝率より「最適な一撃機会」を重視する
Never Do: 不利地形での追撃戦
Combat Philosophy: 待機・見切り・一撃離脱
Relationship Bias: 無駄を嫌う指揮に同調

## Character Logic

Name: バーサーカー（ヘラクレス）
Identity: 高耐久高圧力の制圧型サーヴァント
Core Desire: 圧倒的火力で短時間に戦線を破砕する
Fear: 制御喪失で味方計画を破壊すること
Secret: 供給魔力が不安定だと暴走リスクが跳ね上がる
Never Do: 魔力不足状態での長期拘束戦
Combat Philosophy: 強圧接敵・面制圧・押し切り
Relationship Bias: 明確で短い命令のみ高精度で反応
